# AI Chat System Database using PostgreSQL

## Overview

This project is a complete PostgreSQL database implementation for an AI-powered chat application. It demonstrates relational database design, normalization, table relationships, SQL queries, and database management concepts used in real-world backend applications.

The database simulates how modern AI chat platforms manage users, conversations, AI models, messages, and user feedback.

---

## Database Schema

The project consists of five interconnected tables:

- Users
- Conversations
- AI Models
- Messages
- Feedback

Each table is connected using Primary Keys and Foreign Keys to maintain referential integrity.

## Database Tables

### Users
Stores user account information.

### Conversations
Stores conversation history for each user.

### AI Models
Stores information about AI models such as GPT, Claude, and Gemini.

### Messages
Stores both user and AI-generated messages.

### Feedback
Stores ratings and feedback given to AI responses.
